Administration Help

This help page explains the administrative functions and how to use them.

  • Options Editor - Maintains the system variables that can change over time.
  • User Management - manages agents and their access levels.
  • Daily File Upload - update the database from the corporate listing.
  • Upload Log Viewer - displays the upload information.
  • DAT Spreadsheet - Generates the uploadable spreadsheet to DAT.
  • Start Tickler Processing - Manages the tickler processing page.

Option Editor

This is a simple editor used to maintain the system level option values. Currently it only contains the trailer length and type definitions.

Opt name is the name for the option

Opt value is the value of the option.

Feel free to add truck length and type values. They are used to populate the dropdown lists on the driver page.

User Management

This page is used to manage access to the application.

Each agent and broker must be defined with a unique id and password combination before they can login and use the application.

Fields:

  • User Id is the users login name.
  • Password is the user's login password. The conbination of id and password must be unique.
  • Access Level defines the type of access the user has.
    • user is the general access level with access to all but the administrative menu items.
    • admin defines an administrator. It allows access the the admin menu in addition to all of the others.
    • broker is used to permit a broker to access the driver list page.
  • List Level is no longer used.
  • User Name is the users name...
  • Office is the office the user is associated with. It is used to filter access to the drivers, etc. that are specific to the office.
  • Phone, Fax and Email are the contact numbers for the user.

The "Find" button lists the users so that you can edit their values.

Daily File Upload

Each day corporate sends out a list of drivers in an RTF formatted file. This file is converted to a text file and uploaded to the site server to update the drivers and loads tables.

Convert the file to text by opening it with Word and saving it as text.

Open the upload page and click the "Browse" button and select the text file.

Click the "Upload" button to upload the file to the server and process it.

The update processing creates new drivers and loads.The driver fields inserted are name, trailer length and type, truck number,telephone, comments, twic and office id.The load fields inserted are pickup and delivery locations, delivery date, driver name, office id and load number.
