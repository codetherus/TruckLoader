The contact management page is used to maintain the contact values for drivers, brokers and agents

Fields:

  • Contact Type:
    • Home - home phone number
    • Cell - cell phone number
    • Fax - fax number
    • Email - Email address
    • Other - any not otherwise defined
  • Entity Type:
    • Agent - a site user
    • Broker - a broker you deal with
    • Driver - a driver you deal with
  • Entity:
    • Agent names if the entity type is "Agent"
    • Broker company names if the entity type is "Broker"
    • Driver Name if entity type is "Driver"
  • Number - The contact value.

Buttons:

  • Find - displays a list of contacts from which you may choose for editing.
    • Select a Contact Type, Entity Type and/or Entity to filter the find.
  • Insert - Add the record displayed.
  • Update - Update the displayed record. Must have been read first.
  • Delete - Remove the displayed user record. Must have been read first.
  • Help - Display this page.
