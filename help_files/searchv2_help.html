This is the help page for the search page.
Use this page to locate a driver.

Fields:

  • Driver Status - you may choose active, inactive or both for display.
  • Search Order determines the display order - driver name or truck number.
  • Agent is a list of agents in your office. CHoose one or nnone to filter only drivers assigned to a particular agent.

When you click the submit button, the search is conducted and any qualifying drivers are listed. Click a driver name to display the driver record.
