The user management page is used to maintain the table of users.
This table controlls access to the web site.

Each user is assigned an id and password as well as a level - administrative, user or broker - and a list view level - administrative or none.

The administrative user has access to the administrative menu items as well as the rest of the site.
The user user sees all of the site without the administrative menu.
The broker user only sees the broker list page.

Buttons:

  • Find - display a list of users that you select from for editing.
    • Leave the User Id field blank for a list of all users.
    • Enter a partial user id for a list of matching user id's.
  • Read - read the record of the user id entered. Looks for an exact match.
  • Insert - Add the record displayed.
    • User Id, Password and Accss Level are required.
    • The user id and password combination must be unique.
  • Update - Update the displayed record. Must have been read first.
  • Delete - Remove the displayed user record. Must have been read first.
  • Reset - Clear the display.
  • Help - Display this page.
